Implementation

This page uses the default configuration with just three required files:

  1. clippy.css - ClippyJS styles
  2. jquery.min.js + clippy.js - ClippyJS library
  3. clippy-chat.min.js - The chat widget

Configuration

The widget is initialized with:

ClippyChat.init({
    apiUrl: 'http://localhost:8001',
    clippyPosition: 'bottom-right'
});
